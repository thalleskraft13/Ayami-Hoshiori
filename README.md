## ➕ Adicionar ao servidor

1. Clique no link de convite abaixo:

   **[👉 Clique aqui para adicionar o Ayami ao seu servidor](https://discord.com/oauth2/authorize?client_id=1441027871069048902&permissions=8&scope=bot%20applications.commands)**

2. Escolha o servidor onde você é administrador e confirme as permissões solicitadas.
3. Pronto! Digite `/` no chat do servidor para ver todos os comandos disponíveis.

### Permissões recomendadas

O Ayami funciona melhor com permissão de **Administrador**, já que vários sistemas (tickets, automações, segurança, cargos automáticos) precisam gerenciar canais, cargos e mensagens. Se preferir um controle mais granular, o mínimo necessário é:

- Gerenciar Cargos, Canais e Mensagens
- Enviar Mensagens, Embeds e Anexos
- Usar Comandos de Barra (Slash Commands)
- Gerenciar Webhooks *(necessário para tickets e automações)*

### Primeira configuração

Depois de adicionar, rode o comando de configuração para deixar o bot no idioma e do jeito que você quiser:

```
/configurar
```

Por ele dá pra ajustar idioma do servidor, canais de log, cargos automáticos e recursos que ficam ativos.

## 🌟 O que o Ayami faz

| | |
|---|---|
| 🧩 **Logic Builder** | Crie automações do seu jeito, sem escrever nenhuma linha de código — tudo por botões e menus do Discord. "Quando X acontecer, faça Y." |
| 📜 **Logic Script** *(Premium)* | Para quem quer ir além do visual: uma linguagem de automação própria, com [documentação completa](https://ayami-hoshiori.discloud.app/docs). |
| 🎟️ **Tickets** | Painéis de atendimento completos, com fila de perguntas, transcrição da conversa e atribuição automática de cargo pra quem atender. |
| 🎉 **Sorteios** | Crie sorteios com requisitos de entrada, autorização de participantes e exportação da lista de ganhadores. |
| 🛡️ **Segurança** | Verificação de novos membros, auto-moderação, canal-armadilha e proteção contra raids. |
| 🎨 **Imagens & Vídeos** | Gere memes, molduras e vídeos personalizados direto no chat. |
| 📊 **Rank de Aventureiro** | Sistema de XP e atividade que recompensa quem mais participa do servidor. |
| 🌐 **Multilíngue** | Suporte a Português, Inglês e Espanhol — cada servidor escolhe o idioma preferido. |
| ✨ **Premium** | Recursos extras para quem quiser ir além do plano gratuito. |

## 🚀 Primeiros passos

Depois de adicionar o bot e rodar `/configurar`, alguns comandos bons pra começar:

- `/logic` — abra o Logic Builder e crie sua primeira automação
- `/embed` — monte embeds personalizados para o seu servidor
- `/sorteio` — crie seu primeiro sorteio
- `/personagens` — veja os personagens disponíveis no sistema de RPG
- `/ayami-info` — veja informações e status do bot
